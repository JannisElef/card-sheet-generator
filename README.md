# Card Sheet Generator

[website](https://janniselef.github.io/card-sheet-generator/) · [more information](https://janniselef.github.io/projects/card-sheet-generator/)


Online-Tool for preparing print sheets for playing cards, refining scans, organizing and slicing them.
Scan or download some cards, retouch faded colors, fix janky alignments, export, print and play with them.
